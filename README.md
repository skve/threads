# @skve/threads

A plugin for adding @radixui/color stripe utilities with Tailwind

## Installation

Install the plugin from npm:

```sh
npm install -D @skve/threads
```

Then add the plugin to your `tailwind.config.js` file:

```js
// tailwind.config.js
module.exports = {
  theme: {
    // ...
  },
  plugins: [
    require('@skve/threads'),
    // ...
  ],
}
```
